总览结论

当前最重要的修正是：早期 report.json 里的 tests_passed/resolved 来自 mini-SWE-agent 退出前的轨迹日志，很多是 action was not executed，不能作为成功/失败标签。现在已经接入官方 SWE-bench harness：99 个 run 写入 official_eval.json，其中 94 completed、40 official resolved、54 official unresolved 可进入 DPO/debug，5 个 requests run 仍 pending。本机 90 个已下载 SWE-bench official latest 镜像也已通过下载闭环审计，当前 missing_mini=0、missing_official=0。

质量结论已改变：旧 filtered DPO v0.1 可作为结构样本参考，但不能直接训练。当前 v0.2 已按官方结果重建：只有 official resolved 进 SFT，只有 official unresolved + gold patch 进 DPO/debug。94 个 completed 官方评测结果已同步回填到 runs/*/report.json 主标签，旧 mini 标签保留在 legacy_* 字段；当前 report_official_mismatch=0。
